## 🎯 项目简介

您好！这是我的个人博客，基于 Hugo 和 GitHub Pages 搭建。

创建本博客的主要目的，是为了记录和整理在备战秋招/春招过程中复习的技术知识点（俗称“八股文”）。我希望通过博客的形式，系统化地梳理知识体系，并将其公开分享，以检验自己的学习成果，最终在求职季中取得理想的 offer。
